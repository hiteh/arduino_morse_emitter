# Contributing

Thank you for considering a contribution to `arduino_morse_emitter`.

## Before you start

Please:

- open an issue for larger changes,
- keep pull requests focused and easy to review,
- describe the expected behavior and the actual behavior for bug fixes,
- update documentation when behavior or usage changes.

## Development guidelines

- keep the Arduino sketch compiling cleanly,
- prefer readable code over clever shortcuts,
- preserve existing behavior unless the change explicitly targets that behavior,
- document non-obvious logic with concise comments.

## Pull request checklist

Before submitting a pull request, verify that:

- the sketch still compiles,
- comments and formatting remain consistent,
- the README is updated when needed,
- the change is scoped to a single clear purpose.

## Discussion

For new features, roadmap ideas, or architectural changes, start with an issue before implementation.
