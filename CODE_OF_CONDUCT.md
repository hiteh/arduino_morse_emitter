# Code of Conduct

This project aims to maintain a respectful, constructive, and welcoming environment.

## Expected behavior

Participants are expected to:

- communicate respectfully,
- focus on ideas rather than people,
- accept constructive feedback,
- help keep discussions relevant and useful.

## Unacceptable behavior

The following behavior is not acceptable:

- harassment, threats, or abusive language,
- discriminatory or demeaning comments,
- deliberate disruption of discussions or review processes,
- sharing private information without consent.

## Enforcement

Project maintainers may edit, remove, or reject content that violates this code of conduct and may restrict participation when necessary to protect the project environment.
